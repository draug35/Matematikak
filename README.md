# Matematikak

Batxilergoko matematikarako baliabide interaktiboak. Helburua ez da ariketa mekanikoak biderkatzea, baizik eta ikasleek irudikapen sinbolikoa, taula, grafikoa, interpretazioa eta testuingurua lotzea.

## Baliabideak

- [`funciones-maquina-grafica.html`](./funciones-maquina-grafica.html): euskarazko sarrera-laborategia funtzio linealetarako. Ikasleek `y = mx + b` eredua manipulatzen dute, aurreikuspenak egiten dituzte, ohiko akatsak zuzentzen dituzte, formula ezkutuak asmatzen dituzte eta testuinguru propioak sortzen dituzte.
- [`laboratorio-funciones/`](./laboratorio-funciones/): funtzioen analisi aurreratuagoa lantzeko laborategia, 2. Batxilergorako. Domeinua, deribatua, kurbadura, ukitzailea, integrala, asintotak eta ez-deribagarritasuna lantzen ditu.

## Nola ireki

Ireki `index.html` nabigatzaile moderno batean. Dena estatikoa da; ez da instalaziorik behar.

## Antolaketa didaktikoa

Repoak bi ibilbide osagarri eskaintzen ditu:

1. Oinarrizko ibilbidea: funtzio linealak ulertu, grafikoak irakurri eta testuinguru errealekin lotu.
2. Ibilbide aurreratua: funtzioen analisia sakondu, kasu bereziak aztertu eta arrazoibidea txosten moduan adierazi.
