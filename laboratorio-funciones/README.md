# Laboratorio visual de funciones

Aplicación HTML local para trabajar análisis de funciones en 2º de Bachillerato. Incluye interfaz bilingüe castellano/euskara y cuestionario adaptado al caso seleccionado.

## Abrir

Abre `index.html` en un navegador moderno. No necesita instalación ni conexión a internet.

## Archivos

- `index.html`: estructura de la aplicación.
- `styles.css`: diseño sobrio y responsive.
- `app.js`: lógica matemática, trazado SVG e interacción.
- `guia-docente.md`: propuesta de uso en clase.
- `actividad-alumnado.md`: ficha breve para el alumnado.

## Enfoque

El laboratorio usa casos curados en lugar de entrada libre de funciones. Esto permite que cada caso tenga una intención didáctica clara: dominio, derivada, asíntotas, curvatura, integral o no derivabilidad.

## Cuestionario

Cada función tiene preguntas propias con pista, corrección y siguiente acción. La intención no es puntuar rápido, sino obligar al alumnado a mirar el dominio, la derivada, la curvatura o la discontinuidad correspondiente antes de concluir.
